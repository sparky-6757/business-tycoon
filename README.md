# HelixBridge Business Tycoon

Publish-ready Vite + React build of the game.

## Local run

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

The production output is generated in `dist/`.

## Deploy settings

- Build command: `npm run build`
- Publish/output directory: `dist`
- Framework: Vite / React
